# UnplugJourney Placeholder Site

A lightweight modern placeholder for the rebuilt UnplugJourney.com website.

## Purpose

This site replaces the old WordPress-style feel with a clean static front door for UnplugJourney while preserving the brand's balanced, practical, family-first positioning.

## Included sections

- Hero section with clear value proposition
- Brand trust strip
- Content pillar preview
- UnplugJourney framework
- Resource hub preview
- Blog/category preview
- Glossary
- FAQ
- Placeholder email signup section

## Brand direction used

The design uses the UnplugJourney brand palette:

- Digital Blue: `#1E5F74`
- Natural Green: `#4BA87D`
- Balanced Teal: `#2A9D8F`
- Warm Sand: `#E9C46A`
- Sunset Orange: `#F4A261`
- Deep Coral: `#E76F51`
- Off-White: `#F8F9FA`

Typography uses Montserrat and Merriweather from Google Fonts.

## Deployment

This is a static site and can be deployed with:

- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages

No build step is required.
